## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] pnpm virtual store path format mismatch**
- **Found during:** Task 2 — first `pnpm build` emitted 0 packages
- **Issue:** Plan assumed metafile input keys start with `node_modules/` (flat npm layout). pnpm's virtual store uses relative paths like `../../../node_modules/.pnpm/<store-entry>/node_modules/<actual-pkg>/file.js`. The original prefix check `input.startsWith("node_modules/")` matched nothing.
- **Fix:** Changed matching to `input.includes("node_modules/")` and switched extraction to find the **last** `node_modules/` segment in the split path. Skip segments where the next part is `.pnpm` (that's the store-entry level, not the actual package). Build the full relative path up to the real package root for `package.json` and `LICENSE` reads.
- **Files modified:** `esbuild.mjs` (within same task, before commit)
- **Commit:** `328c3c3` (fix folded into the task commit)
