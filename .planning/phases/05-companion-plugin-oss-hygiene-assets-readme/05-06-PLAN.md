**Capture Setup:**
1. Open VS Code with the extension loaded (F5 debug or installed VSIX)
2. Open Discord desktop client, ensure you're online
3. Ensure Discord sidebar shows your user status
4. Open a screen recorder positioned to capture the Discord sidebar + VS Code terminal

**Capture Sequence (15-30 seconds):**
1. [0-3s] Show Discord sidebar with "Idling" or current status
2. [3-5s] Switch to VS Code, open integrated terminal

4. [10-15s] Show Discord sidebar flipping to AGENT_ACTIVE with goblin copy
5. [15-20s] Let it cycle through one or two rotation ticks
6. [20-25s] Exit claude session (Ctrl+C or `/exit`)
7. [25-30s] Show Discord sidebar returning to previous state

**Post-Processing:**
```bash
ffmpeg -i demo.mp4 -vf "fps=10,scale=720:-1:flags=lanczos" -c:v gif assets/demo-raw.gif

ls -lh assets/demo.gif  # Must be under 8 MB
```
